# tic_tac_toe
Tic Tac Toe – Python

This is a basic Tic Tac Toe game made using Python. It runs in the terminal and allows two players to play by taking turns on a 3×3 board. The game checks for a winner after every move and also detects a draw.

Features

Two-player game (X and O)

Simple command-line interface

Checks win and draw conditions

Prevents invalid moves
